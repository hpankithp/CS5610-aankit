Main Components of the Project :


This project has been built on ASP .NET platform. Here the concept of Master Page has been used to reduce the repeared code.

1. Login and Profile Page : Login and Profile pages contains scope for Login and Sigup pages. On Successful login user can see their profile and the popular actors that they have book marked.

2. Get Parameter from URL : There is a javascript file named URLParam that process the query parameter in the URL. There is a global function in the Page that listens to key stroke and on pressing enter key that value is retrieved from the URL and passed to the calling function.

3. HTML pages : HTML pages like Home,Movies,TvShows show 20 most popular elements in their respective field. Each of these pages contains the link to their respective details pages. For Ex : From home you can click on Actors to get the Actor Details. From Movies you get Movie Details and from TvShows you get details about the TVShows.The search box will also open searchPage.aspx along with the search string as query parameter

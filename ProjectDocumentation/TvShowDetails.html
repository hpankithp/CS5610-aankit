Tv Show Details Page Description :


Summary of the Page : This is the Tv Show Details Page of the project. The .js file for this page contains an Ajax call to themoviedatabase API. The Ajax call is made to get the information about the Tv Show such as the Tv Show image ,id , Description, Cast and other attributes about the Tv Show. Id of the Tv Show is passed as a query parameter to tvShowDetails.aspx. This displays the additional infomation about the Tv Show. Here query parameter id which is equal to the id of the desired Tv Show is passed using URLParam.js. Default value is set if the id for the query parameter is not present in URL. Based on the Id ajax call is being made to get the details about it. If any information about the Tv Show or its cast is not available then a default value is set.Each actor can be clicked to get more details about the actor.Also the header contains all the tabs and footer contains the copyright information.

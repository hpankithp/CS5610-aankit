TV Shows Page Description :


Summary of the Page : This is the TV Shows Page of the project. It is one of the NavBar items.The .js file for this page contains an Ajax call to themoviedatabase API. The Ajax call is made to get the list of most popular TV Shows. Each JSON Object contains information about the TV Shows such as the TV Shows image ,id and other attributes about the TV Show. Id of the TV Show is passed as a query parameter to tvShowDetails.aspx. This displays the additional infomation about the TV Show. TV Show Images are appended to a div tag hence they are downloaded from API only once. Also the header contains all the tabs and footer contains the copyright information.

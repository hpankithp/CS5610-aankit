Actor Details Page Description :


Summary of the Page : This is the Actor Details Page of the project. The .js file for this page contains an Ajax call to themoviedatabase API. The Ajax call is made to get the information about the Actor such as the Actor image ,id , Description, and other attributes about the Actor. Id of the Actor is passed as a query parameter to Actor.aspx. This displays the additional infomation about the Actor. Here query parameter id which is equal to the id of the desired Actor is passed using URLParam.js. Default value is set if the id for the query parameter is not present in URL. Based on the Id ajax call is being made to get the details about it. If any information about the Actor or its movie is not available then a default value is set. In the details tab actor details along with the movies gets populated. Each movie can be clicked to get more details about the actor. Also the header contains all the tabs and footer contains the copyright information

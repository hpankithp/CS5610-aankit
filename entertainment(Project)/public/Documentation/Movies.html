Movies Page Description :


Summary of the Page : This is the Movies Page of the project. It is one of the NavBar items.The .js file for this page contains an Ajax call to themoviedatabase API. The Ajax call is made to get the list of most popular movies. Each JSON Object contains information about the movies such as the movie image ,id and other attributes about the movie. Id of the movie is passed as a query parameter to movieDetails.aspx. This displays the additional infomation about the movie. Movie Images are appended to a div tag hence they are downloaded from API only once. Movie poster gets populated in the div.Also the header contains all the tabs and footer contains the copyright information.

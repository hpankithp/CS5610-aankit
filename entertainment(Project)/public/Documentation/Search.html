Search Page Description :


Summary of the Page : This is the Search Page of the project. The .js file for this page contains an Ajax call to themoviedatabase API. This page is called when an actor is searched in the search box provided in the header. The ajax call to the themoviedb.org API returns the information in terms of pages. Hence the page contains the next and the previous buttons to navigate through the pages. The previous button is not available only when the user is on first page and just in case if the query returns only one page then both the next and previous buttons will disappear.Also if the page is the last page in the search results then the next button is not displayed.Each actor can be clicked to get more details about the actor.Also the header contains all the tabs and footer contains the copyright information.

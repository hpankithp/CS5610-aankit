Introduction :


This web page is being built to show details about Movies and TVShows. Project showcases the information about individual actors in each of Movies and TVShows. Whole porject has been made responsive.I have created following different pages :

1. Home Page : This page display popular actors. Also most popular movies are in the form of carousel.Each actor text gets you to another page containing more details about the actor

2. Movies : This page contains popular movies wiht their image as thumbnails. Each movie text gets you to another page containing more details about the movie

3. TvShows : This page contains popular TvShows wiht their image as thumbnails.Each TvShow text gets you to another page containing more details about the TvShows

4. SearchPage : All Pages contains search box to search the actors.

5. Contacts : This page contains information about the location of my company.

6. Actor : This page contains details about the Actor and all the movies he has acted in. From the movies list, one can navigate to movie details page for that particular movie

7. MovieDetails : This page contains details about the Movies and all the star cast for that particular movie. From the cast list, one can navigate to actor details page.

8. TvShowDetails : This page contains details about the TvShows and all the star cast for that particular show. From the cast list, one can navigate to actor details page.

9. Profile Page : This page display the tagged or the bookmarked popular actor for a particular user.User can logout from the profile page and it gets redirected to the home page.

10. Login Page : This page is used for login.After successful login it gets redirected to profile page. This authentication has been done using Passport JS

11. SignUp Page : This page is used for Signup.After successful Signup it gets redirected to profile page. MongoDB has been used for the backend connection.

Movie Details Page Description :


Summary of the Page : This is the Movie Details Page of the project. The .js file for this page contains an Ajax call to themoviedatabase API. The Ajax call is made to get the information about the movie such as the movie image ,id , Description, Revenue and other attributes about the movie. Id of the movie is passed as a query parameter to movieDetails.aspx. This displays the additional infomation about the movie. Here query parameter id which is equal to the id of the desired movie is passed using URLParam.js. Default value is set if the id for the query parameter is not present in URL. Based on the Id ajax call is being made to get the details about it. If any information about the movie or its cast is not available then a default value is set.Each actor can be clicked to get more details about the actor.Also the header contains all the tabs and footer contains the copyright information.

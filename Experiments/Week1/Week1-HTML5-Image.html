This page demonstrates the use of HTML 5 tags (Header Tag used here)

Time :

This tag defines human readable time.
It's snowing heavily at 9:00 AM (Time Tag used here)

DateTime :

DateTime is an attribute in Time Tag that represents machine readable date/time.

It's snowing heavily on Saturday

Figure :

Images and captions can be grouped together using this tag

Figcaption :

Figcaption defines the caption for the image

Code Snippet :
<figure> <img src="../Images/Yosemite_USA.JPG" height="200" width="200" /> <figcaption> Yosemite,USA </figcaption> </figure>

Details :

This tag defines additional details that the user can view or hide

Click here to check details

Defines additional details that the user can view or hide

Code Snippet :
<details > <summary>Click here to check details <summary> <p>Defines additional details that the user can view or hide </p> <details>

Text Formatting

HTML defines special elements, for defining text with a special meaning HTML uses elements like <b> , <i> and <u> for formatting output, like bold , italic and underline text.

<b> : This is Bold text
<i> : This is Italic text
<u> : This is Underline text

Lists

An unordered list starts with the <ul> tag. Each list item starts with the <li> tag. The list items will be marked with bullets

Code Snippet : Unordered List : <ul> <li>Stack</li> </ul>

Unordered List
  • Stack
  • Queue
  • LinkedList
Ordered List

An ordered list starts with the <ol> tag. Each list item starts with the <li> tag. The list items will be marked with numbers.

  1. First
  2. Second
Description List

A description list, is a list of terms, with a description of each term. The <dl> tag defines a description list. The <dt> tag defines the term , and the <dd> tag defines the description.
